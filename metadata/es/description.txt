Nota: F-Droid requiere privilegios de root para instalar Privileged Extension
como un sistema "priv-app".

https://f-droid.org/packages/org.fdroid.fdroid puede hacer uso de los privilegios o permisos del sistema
para instalar, actualizar y eliminar aplicaciones por su cuenta. La única manera de
obtener esos privilegios es convertirse en una aplicación del sistema.

Aquí es donde entra en juego la Extensión Privilegiada: ser una aplicación separada
y mucho más pequeño, se puede instalar como una aplicación del sistema y comunicarse
con la aplicación principal a través de AIDL IPC.

Esto tiene varias ventajas:

* Reducción del uso del disco en la partición del sistema
* Las actualizaciones del sistema no eliminan F-Droid
* El proceso de instalación en el sistema a través de la raíz es más seguro
