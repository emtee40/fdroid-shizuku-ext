Note : F-Droid requiert les privilèges root pour installer Privileged Extension
comme une "priv-app" système.

https://f-droid.org/packages/org.fdroid.fdroid peut utiliser les privilèges systèmes 
pour installer, mettre à jour et supprimer des applications.
Le seul moyen d'obtenir ces privilèges est d'être reconnu comme une application système.

C'est là que l'Extension Privilégiée fait son entrée, en tant qu'application séparée et bien plus petite,
elle peut être installée comme une application système et communiquer avec l'application principale via AIDL IPC.

Ce qui présente de nombreux avantages :

* Réduction de l'espace disque sur la partition système
* Les mises à jour du système ne suppriment pas F-Droid
* Le processus d'installation sur le système via root est plus sûr.

Au lieu de cette version, la plupart des utilisateurs préféreront installer le fichier ZIP de mise à jour "Over-The-Air" (OTA)
depuis https://f-droid.org/packages/org.fdroid.fdroid.privileged.ota.
Source aussi à utiliser pour mettre à jour Privileged Extension lorsqu'il a été installé via le ZIP OTA.
